# 🏋️‍♀️ Academia CrossFit API

Sistema de gestão de atletas e categorias para uma academia de CrossFit, construído com **FastAPI**, **SQLite** e **Alembic**.

## 🚀 Tecnologias utilizadas

- Python 3.11+
- FastAPI
- SQLite
- Alembic
- SQLAlchemy
- Uvicorn
